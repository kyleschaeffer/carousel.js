Simple example

In this example, we’ll use all the default options from carousel.js to create a simple touch-friendly carousel.

<div class="my-simple-carousel">
  <ul>
    <!-- items -->
  </ul>
</div>

<script>
  $('.my-simple-carousel').carousel();
</script>

Hero carousel example

Creating a hero carousel is easy with carousel.js. We’ll enable a few options when initializing our script to add some new functionality to the carousel.

<div class="my-hero-carousel">
  <ul>
    <!-- items -->
  </ul>
</div>

<script>
  $('.my-hero-carousel').carousel({
    paging: true,
    navigation: true,
    loop: true,
    autoplay: true,
    delay: 5000
  });
</script>

Product spinner example

The carousel.js script includes styles for building multi-column carousels, like shown in the example below. Simply add a CSS class, such as sm-4cols, to your carousel HTML in order to display carousel items in multiple columns on different screen sizes.

<div class="my-product-spinner xs-2cols sm-4cols md-5cols lg-6cols">
  <ul>
    <!-- items -->
  </ul>
</div>

<script>
  $('.my-product-spinner').carousel({
    paging: true
  });
</script>
